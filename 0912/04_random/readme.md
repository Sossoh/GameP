# 범위 내의 난수 생성

## 개념 정리
- 난수 생성 함수: `rand()` → 0 이상 `RAND_MAX` 이하의 정수 반환
- 특정 범위의 난수를 얻으려면 **나머지 연산자 `%`와 덧셈/뺄셈**을 조합
- 실행 시마다 다른 값을 얻으려면 `srand((unsigned)time(NULL));`으로 **시드(seed)** 초기화 필요
