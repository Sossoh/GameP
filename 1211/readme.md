# 🎮 똥 피하기 게임 (Ddong Avoid) - Remastered

Unity로 제작한 2D 아케이드 회피 게임입니다. 하늘에서 떨어지는 장애물을 피하고, 다양한 아이템을 활용하여 최고 점수를 갱신하는 것이 목표입니다.

## 🛠️ 개발 환경 (Tech Stack)
* **Engine:** Unity 2021.3.45f2
* **Language:** C#
* **Platform:** PC (Windows/Mac)

## ✨ 주요 기능 (Key Features)

### 1. 게임 플레이 (Gameplay)
* **회피 시스템:** 하늘에서 떨어지는 '먼지(장애물)'를 피해야 합니다.
* **HP 시스템:** 총 3칸의 체력이 존재하며, 장애물 충돌 시 감소합니다.
* **스코어 시스템:** 생존 시간 및 별 아이템 획득 시 점수가 오르며, `PlayerPrefs`를 통해 최고 점수(Best Score)가 저장됩니다.

### 2. 아이템 시스템 (Item System)
확률에 따라 다양한 효과를 가진 아이템이 생성됩니다.
* **⭐ 별 (Star):** 획득 시 점수 대폭 추가 (+100점).
* **❤️ 하트 (Heart):** 감소된 체력을 1칸 회복합니다.
* **🛡️ 방패 (Shield):** 일정 시간(5초) 동안 캐릭터가 반투명해지며 **무적 상태**가 됩니다.

### 3. 연출 및 사운드 (Effect & Sound)
* **파티클 이펙트:** 별 아이템 획득 시 반짝이는 파티클 효과가 발생하여 타격감을 줍니다.
* **BGM/SFX:** 배경음악과 더불어 점프, 피격, 아이템별(별, 하트, 방패) 고유 효과음을 적용했습니다.

## 📝 개발 일지 (Dev Log) & 업데이트 내역

### 🚀 Ver 2.0 업데이트 (최근 수정 사항)

**1. 아이템 상호작용 및 피드백 강화**
* **무적 아이템(Shield) 구현:** 희귀 확률로 등장하는 방패 구현. 획득 시 캐릭터 스프라이트 투명도 조절 및 충돌 무시 로직 추가.
* **시각적 피드백:** 별 획득 시 `Particle System`을 활용한 폭발(반짝임) 이펙트 추가.
* **청각적 피드백:** 기존 단일 효과음을 `Shield`, `Star`, `Heart` 각각 별도의 `AudioClip`으로 분리하여 상황에 맞는 소리 출력. BGM(배경음악) 루프 재생 적용.

**2. UI 및 로직 버그 수정 (Troubleshooting)**
* **Missing Prefab 오류 해결:** Hierarchy 상에서 연결이 끊긴(빨간색) 프리팹들을 Unpack 및 재연결하여 수정.
* **재시작 버그 수정:** `GameManager`에서 게임 재시작 시 `GameOverPanel`이 사라지지 않던 문제 해결 (패널 비활성화 코드 추가).
* **태그(Tag) 오류 수정:** Shield 아이템 충돌 시 발생하던 `Tag definition` 에러 해결 및 태그 등록.
* **코드 리팩토링:** `Player.cs` 내 중복된 충돌 처리 로직(`OnTriggerEnter2D`)을 통합하고 가독성 개선.
